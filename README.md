# rostand.dev

Personal portfolio & blog — built with Next.js 16, React 19, TypeScript, and Tailwind CSS v4.

Bilingual (EN/FR), dark/light theme, fully accessible (WCAG 2.1).

## Stack

- **Framework** — Next.js 16 (App Router, Turbopack)
- **Styling** — Tailwind CSS v4, CSS custom properties
- **i18n** — next-intl (EN/FR, auto-detection)
- **Blog** — MDX, rehype-pretty-code (Shiki)
- **Analytics** — Microsoft Clarity
- **Hosting** — Vercel

## Getting Started

```bash
npm install
npm run dev
```
